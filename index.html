هيكلة بيانات النماذج اللغوية

هندسة التلقين • مسارات RAG • استراتيجية تحليلية

ركز هذا المشروع على تأسيس مسار توليد معزز بالاسترجاع (RAG) لقاعدة بيانات أكاديمية. الهدف كان تدريب النموذج على استخلاص معلومات من وجهات نظر متنوعة، علمية، وغير سائدة.

تضمن دوري البحث التحليلي العميق، ورسم الخرائط المنطقية، وتنفيذ هندسة التلقين (Prompt Engineering) لضمان تقديم الذكاء الاصطناعي لمخرجات دقيقة وذات أبعاد متعددة.

الأرشيف الرقمي النيو-ريترو

Vibe Coding • جماليات Y2K • نشر الويب

محفظة تفاعلية تم بناؤها بالكامل باستخدام HTML و CSS و JS، ونشرها على GitHub Pages. يمثل هذا المشروع مساحة للتجريب البصري ودمج أنظمة الشبكات الحديثة مع جماليات حقبة Y2K.

تعتمد الواجهة بشكل كبير على متغيرات CSS، والدمج غير التقليدي للخطوط، وحركات الواجهة التي تتحدى التصميم الشبكي المتماثل المعتاد.

تحليل السرديات السينمائية

مونتاج فيديو • استراتيجية محتوى • الازدواجية النفسية

تحليل استراتيجي شامل للازدواجية النفسية في السينما الحديثة. تضمن هذا المشروع استهلاكاً تحليلياً عميقاً للوسائط المتعددة، وتُرجم إلى مقال مرئي (Video Essay) تفاعلي.

باستخدام برامج مثل DaVinci Resolve، تم تحسين عملية المونتاج تقنياً لخدمة إيقاع السرد، مع دمج تقنيات القص المرئي متعددة الطبقات مع نتائج البحث الأكاديمية.
